== Description ==

This plugin will display up to 30 of your magazines or publications that are hosted on ISSUU using cover images.

== Installation ==

1. First download this plugin to your desktop.
2. Extract the Plugin folder to your desktop.
3. Read through this "readme" file thoroughly to ensure you follow the installation instructions.
4. Using your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
5. Inside your WordPress admin area, head to the Plugins screen and find the newly uploaded Plugin in the list.
6. Click on Activate Plugin to activate it.
7. Click on the Settings link under the plugin name to enter your API key and secret and click save
8. Copy and paste the shortcode [display_magazines] into any page where you would like the magazine listing to display

== Frequently Asked Questions ==

= How do I generate an API key =

This plugin requires an ISSUU API key & secret to work. You can generate one by visiting https://issuu.com/home/settings/apikey

= Ok, now what? =

After logging in, click on the green button that says 'CREATE API KEY'. You will need to copy the API key and API secret into each of the corresponding fields on the plugin settings screen.
